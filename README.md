# TikTok爬虫
* 更新时间2021/4/24  抖音version15.2.0
* 运行环境: Win10、Python3.7
---
## 食用方式
```
1. 安装模块
pip install -r requirements.txt
2. 修改settings.py配置文件参数详见文件
3. 运行main.py
```
---
## 文件内容
|  文件名   | 功能  |
|  ----  | ----  |
| reptile  | 爬虫代码文件夹 |
| api  | api地址|
| data  | 数据持久化|
| main  | 运行接口|
| settings  | 配置文件(请求headers)|
| tiktok_reptile  | 数据库文件|
| requirements.txt | 项目依赖模块 |
| tiktok_reptile.sql | 数据库表结构导出|
